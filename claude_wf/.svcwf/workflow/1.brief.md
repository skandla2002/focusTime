# Brief

> **AI 처리 규칙**: `## 입력` 섹션의 내용을 읽고 `plan.md`에 이슈를 추가한 뒤, 해당 내용을 이 파일에서 삭제한다.

## 사용 방법

`## 입력` 아래에 자유롭게 요청·아이디어·배경을 적으세요. 형식 없이 쓰면 됩니다.
AI가 읽고 분석하여 `.workflowSvc/workflow/2.plan.md`에 이슈를 추가합니다.
처리 완료 후 입력 내용은 자동으로 삭제됩니다.

**잘 쓰는 예시:**

- 로그인 화면 만들어줘. Google OAuth로 하고, 로그인 상태는 DataStore에 저장해.
- 홈 화면 RecyclerView가 스크롤할 때 버벅거려. Paging3 적용이 필요할 것 같아.
- Room DB에서 데이터 조회가 느려. 쿼리 최적화나 인덱스 추가가 필요한지 봐줘.
- 알림 기능 추가해줘. 매일 오전 9시 WorkManager로 알림 트리거.
