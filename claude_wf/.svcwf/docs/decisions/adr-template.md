# Architecture Decision Records (ADR)

주요 기술/설계 결정 사항을 기록합니다.
"왜 이렇게 만들었는가"를 미래의 팀원과 AI가 이해할 수 있도록 남깁니다.

---

## ADR 작성 형식

```markdown
## ADR-NNN: [결정 제목]

- **날짜**: YYYY-MM-DD
- **상태**: 제안 / 승인 / 폐기
- **배경**: 이 결정이 필요했던 이유
- **결정**: 선택한 방향
- **근거**: 선택 이유
- **대안**: 고려했으나 선택하지 않은 옵션과 그 이유
- **결과**: 이 결정으로 인한 영향 (장단점)
```
