# Brief

> **AI 처리 규칙**: `## 입력` 섹션의 내용을 읽고 plan.md에 이슈를 추가한 뒤, 해당 내용을 이 파일에서 삭제한다.
> 입력 영역 외 다른 섹션(안내, 예시, 요청 목록)은 수정하지 않는다.

## 사용 방법

`## 입력` 아래에 자유롭게 요청·아이디어·배경을 적으세요. 형식 없이 쓰면 됩니다.
AI가 읽고 분석하여 `.workflowSvc/workflow/2.plan.md`에 이슈를 추가합니다.
이슈 추가시 
처리 완료 후 입력 내용은 자동으로 삭제됩니다.

**잘 쓰는 예시:**

- 로그인 기능 추가해줘. JWT 방식으로 하고 만료는 1시간. refresh token은 어떻게 할지 아직 결정 못했어. 기존 auth 관련 코드가 있으면 확인해봐.
- 결제 모듈에서 간헐적으로 금액 계산 오류가 나. 특히 할인 쿠폰이랑 겹칠 때. 로그 보면 discount 함수 쪽인 것 같아.
- 대시보드 페이지 로딩이 너무 느려. API 호출을 줄이거나 캐싱을 써야 할 것 같은데 어떤 방식이 좋을지도 같이 판단해줘.
