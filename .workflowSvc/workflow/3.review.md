```markdown
- [ ] 커밋 완료 / 커밋 메시지 (.workflowSvc/ai/workflow-rules.md 참조):
  > 1번줄: [PROJ-000] Jira 제목  ← Jira 있을 때, 없으면 생략

  > 한글: 변경 내용 요약
  >
  > English: Summary of changes
```
