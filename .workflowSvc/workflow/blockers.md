# Blockers

현재 작업을 막고 있는 이슈와 의존성을 추적합니다.

## 상태 정의

- 🔴 **Blocked**: 해결 전까지 진행 불가
- 🟡 **At Risk**: 해결 필요하나 일단 진행 가능
- 🟢 **Resolved**: 해결 완료

## Blocker 기록 형식

```
## [번호] [제목]
- **상태**: 🔴 / 🟡 / 🟢
- **발생일**: YYYY-MM-DD
- **내용**: 무엇이 막혀있는가
- **의존 대상**: 누구/무엇이 해결해야 하는가
- **해결일**: YYYY-MM-DD (해결된 경우)
- **해결 방법**: (해결된 경우)
```
